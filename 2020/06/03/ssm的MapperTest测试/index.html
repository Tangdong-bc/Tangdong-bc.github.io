​ Caused by: org.springframework.beans.factory.BeanCreationException:

​ Error creating bean with name ‘sqlSessionFactory’ defined in class path resource [applicationContext.xml]: Invocation of init method failed; nested exception is org.springframework.core.

​ NestedIOException: Failed to parse mapping resource: ‘file [D:\JAVAEE workplace\ssm-crud\target\classes\mapper\EmployeeMapper.xml]’; nested exception is org.apache.ibatis.builder.

​ BuilderException: Error parsing Mapper XML. Cause: org.apache.ibatis.builder.

​ BuilderException: Error resolving class. Cause: org.apache.ibatis.type.

翻译：

​ 无法加载ApplicationContext。

​ 造成原因：在内路径applicationContext.xml下创建sqlSessionFactory的bean出错：调用init方法失败，嵌套异常为org.springframework.core。

​ NestedIOException:无法分析映射资源 EmployeeMapper.xml。

​ BuilderException:映射文件出错。

​ BuilderException: 解析类时出错。

​ TypeException: 无法解析类型别名 ‘com.atguigu/crud/bean/Department’，造成原因找不到类com.atguigu/crud/bean/Department。
